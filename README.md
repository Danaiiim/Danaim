# Данайым Капесова — Portfolio

Личный сайт-портфолио, созданный вручную на HTML, CSS и JavaScript в рамках конкурса CapEducation «Обо мне».

## Содержание сайта
| # | Секция | Описание |
|---|--------|----------|
| 01 | О себе | Имя, факты, стек |

## Технологии

- HTML5 — семантическая разметка
- CSS3 — Flexbox, Grid, медиазапросы, CSS-анимации
- JavaScript  бургер-меню, плавный скролл, Intersection Observer

## Запуск

 Live Server в VS Code.

## Структура проекта

```
portfolio/
├── index.html      # Разметка сайта
├── style.css       # Все стили
├── script.js       # JavaScript интерактив
└── README.md       # Этот файл
```

## Особенности

- Полностью адаптивный дизайн 
- Плавный скролл между секциями
- Анимации появления элементов при скролле 
- Подсветка активного пункта навигации
- Без фреймворков и библиотек — чистый код
